# Laboratorio 2 - Sistemas Embebidos (2026-1)

## 📌 Objetivo
Diseñar e implementar una aplicación básica en un microcontrolador **ESP32** utilizando una **matriz de LEDs 8x8 bicolor (24 pines)**.  
El proyecto consiste en desarrollar un **videojuego sencillo**, con entradas mediante pulsadores y uso de al menos dos colores de la matriz.

---

## ⚙️ Procedimiento
1. Diseñar el videojuego (lógica y mecánica a criterio del desarrollador).
2. Construir el circuito electrónico con ESP32 + matriz de LEDs + pulsadores.
3. Programar el firmware en **C (ESP-IDF)** sin librerías externas.
4. Implementar y probar el montaje físico.
5. Verificar el correcto funcionamiento del videojuego.
